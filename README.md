# Craft:
To craft a **Temporal Bottle** you will need:
- 1 **Clock**
- 3 **Gold Ingot**
- 1 **Lapis Lazuli**

![Temporal Bottle Craft](doc/1.0%20-%20Temporal%20Bottle%20Craft.png)

# Collecting Time:
Keeping the **Temporal Bottle** in your inventory will allow it to start collect time from the world that can be used later.

![Temporal Bottle collecting time](doc/2.0%20-%20Time%20in%20Bottle.gif)

# Cost to Accelerate:
For every doubling of speed, the **total time cost also doubles**.
Below is a list of how much time you need to accelerate to a certain level of speed.

![Temporal Bottle List of Cost](doc/3.0%20-%20Temporal%20List.png)

# Showing How Accelerate:
To accelerate some block you **need** to have **time**.
Using the **Temporal Bottle** in your hand and interacting with the blocks below will create a time zone 2 times faster, interacting again now it'll be 4 times faster.

![Accelerate Furnace](doc/4.0%20-%20Accelerate%20Furnace.gif)
![Accelerate Brewing Stand](doc/4.1%20-%20Accelerate%20Brewing%20Stand.gif)

# Commands:
If you don't want to wait for the **Temporal Bottle** to collect time, you can use the **"/givetime"** command.
You can:
* Set Time: **"/givetime set [seconds: number]**
* Add Time: **"/givetime add [seconds: number]**
